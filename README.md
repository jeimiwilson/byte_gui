# ByteGui: CLI para leer ByteBeat

## Requerimientos

1. Que no exista ya
2. Correr en Termux y Linux
3. Tener una interface sencilla (gráfica)
4. Tener melodías ya cargadas y poder agregar otras
5. Dejar espacio para mejoras (tipo guardar músicas)
